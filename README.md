🇯🇵 Japanese Holiday Calendar / 日本の祝日カレンダー
A simple web application that displays Japanese national holidays. Built with React, Vite, and Tailwind CSS.

日本の祝日を表示するシンプルなウェブアプリケーションです。React、Vite、Tailwind CSSで構築されています。

🧩 Features / 特徴
Displays a list of Japanese national holidays.

Responsive design using Tailwind CSS.
